# NexaTech Digital – Backend

This is the backend service for the NexaTech Digital IT Services website. It handles form submissions, stores enquiries in the database, and sends email notifications to the admin.

---

## 🚀 Live API
[https://sap-backend-enquiry.onrender.com/]

---

## 🧠 Project Overview

The backend is built using MVC architecture and is responsible for:
- Receiving contact form submissions
- Storing enquiries in MongoDB Atlas
- Sending email notifications using Nodemailer
- Providing RESTful APIs for frontend integration

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Nodemailer
- MVC Architecture

---

## 📦 Features
- REST API for Contact Form
- MongoDB database integration
- Email notifications on new enquiry
- Clean MVC folder structure
- Environment-based configuration support
